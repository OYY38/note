# charles

# 1. 电脑端抓包

电脑charles客户端下载ssl证书 Help -> SSL Proxying -> Install Charles Root Certificate

安装证书

选择本地计算机

选择将所有的证书放入下列储存,点击浏览,然后选择受信用的根证书颁发机构

勾选 Windows Proxy
